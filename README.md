# 🌌 Acerca del proyecto

**COVID19 SL** es una aplicación móvil enfocada en mostrar estadísticas y datos relacionados con el COVID-19 en Sri Lanka.

La aplicación permite:

- 🦠 Visualizar casos COVID-19
- 📊 Consultar estadísticas en tiempo real
- 🌐 Monitorear información nacional
- 📱 Acceder desde dispositivos móviles
- 📈 Ver tendencias y reportes
- 🔔 Mantener seguimiento de datos sanitarios

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/usuario/COVID19_SL.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd COVID19_SL
```

---

## 3️⃣ Instalar dependencias

```bash
flutter pub get
```

---

## 4️⃣ Ejecutar aplicación

